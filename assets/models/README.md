Drop high-quality `.glb` vehicle models in this folder to upgrade the visuals without changing game code.

Suggested names:
- `zipsy.glb`
- `traffic-sport.glb`
- `traffic-suv.glb`
- `traffic-coupe.glb`
- `city-tower-a.glb`
- `city-tower-b.glb`
- `city-billboard.glb`

The game currently falls back to procedural cars when these files are not present.
